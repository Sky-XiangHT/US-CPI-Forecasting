## Project Overview

This project aims to forecast U.S. inflation (CPI) using modern econometric methods and machine learning models.

The goal is to compare traditional econometric approaches with modern machine learning algorithms in macroeconomic forecasting.

---

## Research Question

Can machine learning improve the prediction accuracy of U.S. inflation compared with traditional econometric models?
